# MSDS684: Reinforcement Learning

A comprehensive coursework repository for **MSDS 684: Reinforcement Learning**. This repository contains weekly lab assignments, notebooks, and implementations covering core reinforcement learning topics.

## Course Structure

### Week 1: The Multi-Armed Bandit Problem & MDP Foundations

**Directory:** [`W1/`](W1/)

Explore the foundations of reinforcement learning through the multi-armed bandit problem.

- **Lab Assignment:** [`MSDS684_W1_Lab.ipynb`](W1/MSDS684_W1_Lab.ipynb) — Hands-on Jupyter notebook implementing bandit algorithms
- **Framework Documentation:** [`W1/README.md`](W1/README.md) — Complete guide to the Python bandit framework
- **Assets:** [`W1/img/`](W1/img/) — Figures and diagrams
- **Source Code:** [`W1/src/`](W1/src/) — Reusable bandit framework implementation

**Topics covered:**
- Custom Gymnasium environments
- ε-greedy exploration
- Upper Confidence Bound (UCB) algorithms
- Performance comparison and visualization
- Foundational MDP concepts (Sutton & Barto, Chapters 2–3)

---

## Getting Started

1. Navigate to the week's directory (e.g., `W1/`)
2. Read the week's README for framework and assignment details
3. Open the Jupyter notebook to work through the lab exercises
4. Refer to the framework documentation for API usage

## Prerequisites

- Python 3.8+
- Jupyter Notebook
- Libraries: `gymnasium`, `numpy`, `matplotlib` (specified in individual week READMEs)
